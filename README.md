# phcrawler

PH Crawler

## Requirements

- python3
- python3-pip
- aria2

## Install

```bash
pip install git+https://github.com/madobet/phcrawler
```
or
```bash
pip install --extra-index-url=https://test.pypi.org/simple/ phcrawler
```

## Uninstall

```bash
pip uninstall phcrawler
```

## Usage

Help: `phcrawler -h` or `./phcrawler.py -h` or `python phcrawler.py -h`
